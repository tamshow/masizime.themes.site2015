ブログのカスタムフィールド

SEO

フィールド名 フィールド値 フィールド説明 備考
タイトル {blog_meta_title} ブログの METAタグ（Title）を設定します。 :search value="0"
キーワード {blog_meta_keywords} ブログの METAタグ（Keywords）を設定します。 :search value="0"
概要文 {blog_meta_description} ブログの METAタグ（Description）を設定します。 :search value="0"
検索エンジン {blog_meta_robots} チェックするとこのブログをインデックスされないようにします。 -
サイトマップ {blog_meta_sitemap} チェックするとこのブログをサイトマップに出力しないようにします。 -
RSS {blog_meta_rss} チェックするとこのブログのエントリーをRSSに出力しないようにします。 -

サイト設定 基本

フィールド名 フィールド値 フィールド説明 備考
サイト ロゴ {siteLogo} サイトのロゴ画像を設定します。（リサイズされません）
サイト タイトル {siteName} サイトのタイトルを文字で表示させる際に設定します。 -
サイト 説明 {siteDescription} ロゴの下に表示する説明文を設定します -
フッターテキスト {siteFooter} フッターのテキストを設定します。 -
ログインヘッダー {loginHeaderImage} ログイン画面のヘッダーを設定します。幅：960px -
